Instructions

Concentration, also known as Match Match, Memeory, Pelmanism, Shinkei-suijaku, Pexeso, or Pairs, is a card game in which all of the cards are laid faced down on a surface and two cards are flipped face up over each turn. The object of the game is to turn over pairs of matching cards.

Description

This is an exercise to create a fun game of memory where you can test your ability to remember where cards are located.
